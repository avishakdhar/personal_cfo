# Personal CFO

Personal CFO is a Flutter-based personal finance management app that helps users track accounts, expenses, transfers, and financial analytics.

## Features
- Account management
- Expense tracking
- Transfer between accounts
- Dashboard analytics
- Spending breakdown charts
- SQLite local database

## Tech Stack
- Flutter
- Dart
- SQLite (sqflite)
- fl_chart

## Screens
- Dashboard
- Accounts
- Add Transaction
- Transfer
- Analytics

## Upcoming Features
- Income tracking
- Budget planning
- Monthly reports
- Export to CSV
